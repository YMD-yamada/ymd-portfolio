プライバシーポリシー

最終更新: 2026-07-22 · 運営: 山田健登（ymd）

本ポリシーは、ymd が公開する Web サイトおよび Web アプリ（ポートフォリオ https://ymd-portfolio-site.pages.dev/ 上に掲載する制作物を含む。以下「本サービス」）における個人情報の取扱いを定めます。

App Store / Google Play 向けの共通法務ページは別サイトでも公開しています: ストア申請用プライバシーポリシー

1. 取得する情報

  • お問い合わせ時に送信されるメールアドレス・氏名・内容
  • アクセスログ（IP アドレス、ブラウザ情報、参照元など。ホスティング事業者が処理する場合があります）
  • 各 Web アプリが独自に収集する情報（ログイン・設定・入力内容など。該当アプリの説明または本ポリシーの範囲内で扱います）

2. 利用目的

  • 本サービスの提供・運営・改善
  • お問い合わせへの回答
  • 不正利用の防止、障害調査
  • 法令に基づく対応

3. 第三者提供・委託

法令に基づく場合を除き、同意なく第三者へ提供しません。配信・ホスティング・解析のため、Cloudflare / Vercel / Netlify / Render / GitHub 等の処理委託先を利用することがあります。

4. Cookie 等

表示モードやセッション維持のため、ブラウザのローカルストレージ / Cookie を用いることがあります。必須でない解析を導入する場合は、本ページを更新します。

5. 開示・削除

請求は ymd.hude@gmail.com または サポートページ から受け付けます。

6. 改定

改定時は本ページの更新日を改めます。
