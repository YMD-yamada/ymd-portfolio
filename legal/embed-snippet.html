各 Web アプリに貼る法務リンク（雛形）

新規 Web アプリを公開するとき、エージェントがフッターへ埋め込む用です。ユーザー操作は不要です。

コピー用 HTML:

<footer class="app-legal">
  <p>本アプリは
    <a href="https://ymd-portfolio-site.pages.dev/legal/privacy.html" rel="noopener">プライバシーポリシー</a>
    ・
    <a href="https://ymd-portfolio-site.pages.dev/legal/terms.html" rel="noopener">利用規約</a>
    に従って公開しています。
    <a href="https://ymd-portfolio-site.pages.dev/legal/support.html" rel="noopener">サポート</a>
  </p>
</footer>

Markdown / テキスト用 URL:

  • https://ymd-portfolio-site.pages.dev/legal/privacy.html
  • https://ymd-portfolio-site.pages.dev/legal/terms.html
  • https://ymd-portfolio-site.pages.dev/legal/support.html
