Portfolio

学びと実験を、少しずつ形にしていく。

レトロゲームの仕組み、身体を動かすこと、コードと工具——関心は広く、深さはまだ途中です。ストアで公開したアプリはいま準備中で、このサイトでは試行錯誤の記録と、これから増やしていく制作物をまとめます。気になることには正直に向き合い、長く続けられる形を探しています。

制作物

GitHub のリポジトリに「Website」（homepage）として公開 URL を書いたものと、Netlify / Vercel / Render / Cloudflare のトークン連携で拾えるサイトを、デプロイ時に自動で集約しています（手動で書いた既定リンクと重複する URL はまとめます）。R-18 制作物は設定で除外しています。

サイト運用・更新について（開発者向け）

一覧データはリポジトリ内の data/apps.json から読み込みます。更新は npm run sync:apps（または CI）で再生成し、コミットして反映してください。設定は config/apps.config.json、ホスト名まわりは config/site.json です。

運用手順は RUNBOOK.md を参照してください。

プロフィール

「幅広い関心」と「まだ途中の制作」を、むりにひとつにまとめずに説明します。

レトロゲームはハードや回路の話から、運動はフォームや記録の話まで、コードは小さな自動化やこのサイトの保守まで——題材はバラバラでも、仕組みを理解して自分の手で試すところが続いています。いまは実験と学びの途中で、成果として一言で説明できるほど整理できているわけではありません。

そのぶん、このページでは誇張しない書き方を心がけています。共同制作・取材・技術相談などはメールへ。X / Instagram / TikTok はアカウント開設後にリンクを載せ、更新の置き場を増やす予定です。
