サイト運用・更新について（開発者向け）

一覧データはリポジトリ内の data/apps.json から読み込みます。更新は npm run sync:apps（または CI）で再生成し、コミットして反映してください。設定は config/apps.config.json、ホスト名まわりは config/site.json です。

運用手順は RUNBOOK.md を参照してください。

プロフィール

専門領域を一つに絞らず、道具と身体と画面のあいだを行き来しています。

レトロゲームは画面の向こう、運動は身体、プログラムは手元、DIYは工作と工具——置き場所はバラバラでも、「なぜ動くのか」に手が伸びる瞬間は共通しています。調べて、試して、戻ってくる。その反復が積み上がると、自然と横断的な視点になります。

制作物は上の一覧、近況や断片は連絡先のリンク先に置くことが多いです。共同制作や技術相談などは、お気軽にご連絡ください。
